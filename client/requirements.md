## Packages
framer-motion | Beautiful page transitions, scroll animations, and interactive elements
react-hook-form | Managing the contact form state efficiently
@hookform/resolvers | Validating the form using the Zod schema from the shared contract

## Notes
- The application uses a strictly dark, modern, high-tech aesthetic with a custom color palette.
- All sections exist on a single landing page with smooth scrolling.
- Contact form posts to `/api/contact` using the existing backend schema.
- Custom fonts (Space Grotesk and Inter) are applied via standard CSS in index.css.
